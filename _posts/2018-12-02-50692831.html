Oracle Java BCL 라이선스가 2018년 7월 부로 `Java SE subscription` 년단위 구독모델로 개편됐습니다.

Oracle 유료화 모델이 변경 되면서 Atlassian Server, DataCenter에 어떤 영향을 주는지 알아봤습니다.

Oracle JDK는 유료인가?

2019년 1월 이후에 릴리즈되는 Oracle Java SE8은 유료 라이선스 없이 없무용, 상업용으로 사용할 수 없습니다.

그러면 그전의 1.8.181은 BCL 계약의 동의하에 무료로 사용할 수 있습니다.

무료로 사용할 수 있는 범위는 Oracle Java SE 8 을 수정하지 않고 그대로 사용하면 무료로 사용할 수 있습니다.

보통 Spring Framework이나 Java를 사용하지 Java SE를 고쳐서 쓰는 경우는 드물다고 생각합니다..

1.8.181 을 패치 없이 사용하거나 구독형 라이선스를 구매해서 사용할 수 있습니다.

Atlassian Oracle Java 지원

Oracle Java를 사용하는 제품은 Jira, Confluence가 있습니다.

무료 사용자는 Java 1.8 패치 지원이 중단되면서 구매를 해야하나? 궁금하실 텐데요.

Atlassian은 Open JDK 지원 계획을 발표 했습니다.

그리고 Beta Version으로 Open JDK를 지원하는 최신 버전의 Jira, Confluence가 릴리즈 됐습니다.

물론 Oracle JDK도 같이 지원 예정입니다.

https://confluence.atlassian.com/jirasoftware/jira-software-7-13-x-release-notes-957981568.html#JiraSoftware7.13.xreleasenotes-openjdk

https://confluence.atlassian.com/doc/preparing-for-confluence-6-13-958443854.html

현재 1.8.181로 사용하고 계시다면 Jira, Confluence 최신 버전이 안정화 될 때 까지 지켜보다가 Open JDK로 운영하시는게 좋을 것 같습니다.

오픈소스컨설팅과 함께면 더욱 좋겠죠?
