오늘은 Salesforce & Jira Connector add-on (by ServiceRocket) 설정에 대해 기술하도록 하겠습니다.

Connector는 Jira와 Salesforce를 통합하여 Salesforce 객체를 Jira 이슈에 연결하고 Salesforce와의 변경 사항을 푸시 및 풀링 할 수 있도록 합니다.


설치되어 있는 소프트웨어 아래와 같습니다:

  • Jira Software v7.12.3
  • Salesforce & Jira Server Connector Plugin version 6.2.6
  • Salesforce Classic edition Winter '19


1.  Jira 이슈와 SalesForce 기회 연결

1. Jira에 이슈 생성: Summary 필드 입력 후 “ Create ” 버튼 클릭


2. 생성된 이슈를 연 후 “ More ” 클릭 후 “ Manage Application Links ” 선택


3. SalesForce에 로그인 후 Jira랑 연결할 기회를 열기. 기회 ID나 (아래 Option 1 참조) 기회 제목을 (아래 Option 2 참조) 복사 후 Jira Manage Application Links 페이지에 붙이고 “ +Add Opportunity ” 버튼 클릭


3.1. Option 1:

기회 ID를 복사 후 Jira Manage Application Links 페이지에 Remote ID 필드에 붙이고 “ +Add Opportunity ” 버튼 클릭


3.2. Option 2:

기회 제목을 복사 후 Jira Manage Application Links 페이지에 Subject 필드에 붙이고 “ +Add Opportunity ” 버튼 클릭


4. 아래 리스트에 기회 추가된 후 “ Pull Changes ” 클릭


5. Sync 성공적으로 완료 되면 Jira 이슈로 이동해서 확인


6. 자세한 냉용은 SF AppLink 를 클릭해서 SalesForce 기회로 이동 후 확인


2.  Jira 이슈에서 SalesForce 기회 생성


1. Jira 이슈 생성: 필수 필드 입력 후 “ Create ” 버튼 클릭


2. 생성된 이슈를 연 후 “ More ” 클릭 후 “ Manage Application Links ” 선택


3. Jira Manage Application Links 페이지에 “ Create New Opportunity from this Issue ” 클릭


4. SalesForce으로 이동 후 생성된 기회를 열기. SalesForce 기회에 있는 Jira issue 링크를 클릭하면 Jira issue로 이동
